# EventRescue — فزعة لمناسبتك 🚨

منصة ويب تربط أصحاب المناسبات الطارئة بمزودي الخدمات والفنيين في مدينتهم — طلبات خدمة، عروض أسعار، وقبول فوري مع معلومات تواصل مباشرة.

## التقنيات

ASP.NET Core MVC (.NET 10) — Entity Framework Core — SQL Server — ASP.NET Identity — Bootstrap 5 — دعم لغتين (عربي/إنجليزي) مع انقلاب اتجاه تلقائي.

## المزايا الرئيسية

- **للمستفيد:** تسجيل حساب، إنشاء طلب بصورة وتاريخ ووقت، استقبال عروض الفنيين والموافقة على الأنسب، إلغاء أو حذف الطلب.
- **لمزود الخدمة:** رادار طلبات مفلتر حسب مدينته وتخصصه، تقديم وسحب العروض، لوحة مهام مع أرشيف وإجمالي الأرباح، رادار طوارئ للفزعات خلال أقل من 24 ساعة بجميع التخصصات، وزر تفعيل التوفر الفوري.
- **للإدارة:** لوحة إحصائيات، إدارة المستخدمين والطلبات والعروض.
- المزود يستطيع استخدام المنصة كعميل أيضاً إذا كان لديه مناسبة خاصة.

## التشغيل

1. عدّل سطر الاتصال في `appsettings.json` حسب سيرفرك (الافتراضي: `localhost\SQLEXPRESS`).
2. شغّل المشروع — **قاعدة البيانات تنبني وتتعبأ بالبيانات التجريبية تلقائياً عند أول تشغيل**:

```
dotnet run
```
